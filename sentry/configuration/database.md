###Database Instance

----------

Sentry allows you to set a specific database instance for Sentry to connect to.

<table>
	<tr>
		<th>Option</th>
		<th>Type</th>
		<th>Default</th>
		<th>Description</th>
	</tr>
	<tr>
		<td>db_instance</td>
		<td>string</td>
		<td>null</td>
		<td>Name of database instance, or null for the default instance.</td>
	</tr>
</table>
